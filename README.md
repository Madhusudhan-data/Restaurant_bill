Green Bucket Mandi – Digital Billing SystemA modern, responsive, and interactive digital billing terminal designed for Green Bucket Mandi. This application allows staff to manage customer orders, calculate bills including GST, and process payments via a simulated digital payment terminal.
🚀 FeaturesDynamic Menu Interface: Displays food and drink items categorized for easy selection.  Real-time Billing Engine: Automatically calculates subtotals, applies 18% GST, and updates the total amount instantly.  Interactive Cart: Users can add or remove items from the order with real-time feedback.  
Digital Payment Terminal: Features a sleek, automated modal for QR code-based UPI payments. Simulated Success State: Includes a visual transition that confirms payment success and generates a transaction token.  Responsive Design: Built with a mobile-first approach using Tailwind CSS for a premium, dark-themed UI.  
🛠️ Technologies UsedFrontend: HTML5, Vanilla JavaScript  Styling: Tailwind CSS  Icons: Integrated SVG graphics for status updates  
💻 How to UseView Menu: The menu is automatically loaded upon page initialization.  Order Items: Click the "Add" button on any menu item to build your cart. 
Manage Cart: Easily remove incorrect items using the "Remove" button within the billing receipt section. 
Checkout: Click the "Pay Now" button to trigger the payment modal.  
Payment: Scan the displayed QR code to complete the transaction.  Built with ❤️ for efficient restaurant billing management.
