# 🥗 Green Bucket Mandi

A modern, responsive, and interactive digital billing terminal designed for Green Bucket Mandi. This application allows staff to manage customer orders, calculate bills including GST, and process payments via a simulated digital payment terminal[cite: 1, 2].

## 🚀 Key Features

* **Dynamic Menu Interface**: Displays food and drink items categorized for easy selection[cite: 1].
* **Real-time Billing Engine**: Automatically calculates subtotals, applies 18% GST, and updates the total amount instantly[cite: 1].
* **Interactive Cart**: Users can add or remove items from the order with real-time feedback[cite: 1].
* **Digital Payment Terminal**: Features a sleek, automated modal for QR code-based UPI payments[cite: 1].
* **Simulated Success State**: Includes a visual transition that confirms payment success and generates a transaction token[cite: 1].
* **Responsive Design**: Built with a mobile-first approach using Tailwind CSS for a premium, dark-themed UI[cite: 2].

## 🛠️ Technologies Used

* **Frontend**: HTML5, Vanilla JavaScript[cite: 1, 2]
* **Styling**: Tailwind CSS[cite: 2]
* **Icons**: Integrated SVG graphics for status updates[cite: 1]

## 💻 How to Use

1. **View Menu**: The menu is automatically loaded upon page initialization[cite: 1].
2. **Order Items**: Click the "Add" button on any menu item to build your cart[cite: 1].
3. **Manage Cart**: Easily remove incorrect items using the "Remove" button within the billing receipt section[cite: 1].
4. **Checkout**: Click the "Pay Now" button to trigger the payment modal[cite: 1].
5. **Payment**: Scan the displayed QR code to complete the transaction[cite: 1].
